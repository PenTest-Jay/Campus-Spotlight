# 📌 Campus Spotlight Website

---

## 📖 Project Overview

Campus Spotlight is a responsive multi-page website designed to showcase campus events, improve student engagement, and provide a centralized platform for event information, registration, and communication.

The website includes interactive features such as event sorting, scroll reveal animations, image gallery modals, a dynamic calendar system, and a contact page layout.

## ⚙️ Features

- 📅 Event management system with automatic sorting by date  
- 🎯 Interactive event cards with navigation handling  
- ✨ Scroll reveal animations for smooth UI experience  
- 📸 Image gallery modal with next/previous navigation  
- 📆 Dynamic calendar highlighting current date  
- 🧭 Fully responsive navigation bar  
- 📍 Contact page with structured form layout  
- 🔐 Staff login page (front-end simulation only)  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox, Grid, Variables, Animations)  
- JavaScript (DOM manipulation, event handling, UI logic)  
- GitHub Pages (deployment/hosting)

## 🚀 How to Run Locally

1. Clone the repository:
```bash
git clone https://github.com/PenTest-Jay/Campus-Spotlight.git
```

2. Navigate into the folder:
```bash
cd Campus-Spotlight
```

3. Open `index.html` in your browser.

---

## ⚠️ Important Notes

- This is a **front-end only project**
- No backend or database is connected
- Form submissions are for UI demonstration only (no data storage)
- JavaScript must be enabled for full functionality
- Fixed GitHub Pages file path issues to ensure proper loading of assets
